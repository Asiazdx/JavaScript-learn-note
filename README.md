## 笔记整体结构
根据Javascript的组成及其难易程度将笔记分成了三大部分：
- **JavaScript初探**：主要介绍ECMAScript基本语法

- **Javascript高阶**：主要介绍开发中经常用到的一些JS高级编程相关知识
